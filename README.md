# Measure Tool v5

## Features
- Calibration (real units)
- Distance measurement
- Angle measurement
- Units: m, cm, mm
- Grid & Snap
- Undo
- Zoom
- Export image

## Workflow
1. Upload image
2. Mode → Calibration
3. Select 2 points
4. Enter real length (meters)
5. Switch to Distance or Angle

## Notes
Distances are accurate after calibration.
